OVERVIEW--
This project implements a two-player Tic-Tac-Toe game entirely in Python. Players alternate turns to mark their chosen positions on the grid, and the program automatically checks for wins, draws, and valid moves. The code is modular, readable, and serves as a great introduction to game logic and user input handling in Python.

KEY FEATURES

1.Interactive, text-based gameplay for two players
2.Smart input validation to prevent invalid moves
3.Automatic win and draw detection
4.Clean and modular Python code
